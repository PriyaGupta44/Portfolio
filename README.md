Welcome to my personal portfolio website! This project showcases my skills, projects, and journey as a web developer. It is built using **HTML** and **CSS**, with a focus on creating a clean, responsive, and user-friendly design.

## ✨ Features

* 📱 Responsive design
* 🎨 Modern and clean user interface
* 👤 About Me section
* 💼 Projects showcase
* 🛠️ Skills section
* 📞 Contact section
* ⚡ Fast and lightweight

## 🛠️ Technologies Used

* HTML5
* CSS

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/portfolio.git
```

2. Open the project folder.

3. Launch `index.html` in your web browser.

No additional setup or installation is required.
